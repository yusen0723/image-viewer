# Image Viewer

一个最小可用的 Chrome 扩展：

- 在网页中点击图片时，弹出站内自定义查看器
- 鼠标在图片上滚轮缩放，在图片外空白区滚轮切图
- 以鼠标所在位置为缩放中心
- 支持拖拽平移
- 支持识别当前图片附近的图片组，并用方向键或顶部按钮切换
- 支持 `Esc`、点击遮罩、点击关闭按钮退出
- 提供简单的白名单 / 黑名单设置页，并可从查看器内直接打开设置
- 设置面板内提供 GitHub Star 支持链接
- 对浏览器页面缩放后的 `visualViewport` 做了适配，避免工具按钮跑出可视区域

## 安装

1. 打开 Chrome 扩展管理页：`chrome://extensions`
2. 打开右上角“开发者模式”
3. 选择“加载已解压的扩展程序”
4. 选择目录：`/Users/yusen/Documents/Playground/xhs-image-zoom`

## 当前实现说明

- 图片组识别目前基于“当前点击图片所在容器”的通用规则，不是针对某个站点私有 DOM 的强绑定适配
- 默认对所有网站启用，可在设置页切换为白名单模式或配置黑名单
- 如果目标站点的 DOM 后续变化，图片点击识别规则可能需要调整
